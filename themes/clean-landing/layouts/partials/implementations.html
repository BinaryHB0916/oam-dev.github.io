See It In Action

KubeVela is a modern application deployment system powered by Open Application Model.

Leveraging Kubernetes as control plane, KubeVela as a platform is runtime agnostice. It allows you to deploy (and operate) containerized workloads, cloud functions, databases, or even EC2 instances with consistent workflow.
